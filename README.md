# Set Up
## Clone This Repository
```
$ git clone https://github.com/emmyobonyo/travel-notes.git
$ cd travel-notes
$ git checkout develpoment
```

## Run Project
```
$ npm install
$ npm start
```

## Run APP in the Broswer
- Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
